# Robot_War
# Project Description
Developed a command-line based robot war game using C++, where multiple robots engage in strategic combat within a simulated environment, featuring movement, attack mechanics, and game logic.

# Key Features
-Built a CLI-based game system to simulate robot battles
-Designed object-oriented classes for robots, weapons, and game environment
-Implemented core gameplay mechanics:
-Robot movement within a grid/map
-Attack and damage system
-Health tracking and elimination logic
-Developed turn-based combat logic for multiple robots
-Applied game rules and win conditions to determine the outcome
-Managed game state using appropriate data structures
